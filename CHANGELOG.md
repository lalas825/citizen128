# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-11

### Added
- **Citizen128 Brand Identity**: Complete rebranding from "Liberty 128" to "Citizen128".
- **Documentation**: Initial `CHANGELOG.md` to track project history.

### Changed
- **Bundle IDs**: Updated all platform identifiers to `com.citizen128.citizen128` (Android) and `com.citizen128.citizen128` (iOS).
- **Directory Structure**: Refactored Android source paths from `com/liberty128` to `com/citizen128`.
- **Git Remote**: Updated origin to point to the new Citizen128 repository standard.
- **Assets**: Replaced app icons and branding assets to align with the new visual identity.

### Fixed
- Stabilized core functionality for the v1.0.0 release.
